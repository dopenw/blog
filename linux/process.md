# 进程相关
1. 使用nohup运行程序，此时会忽略SIGHUP信号
2. 当控制终端接口检测到调制解调器断开（直接关闭终端，不是exit）或网络已经断开连接，则将挂断信号发送至控制进程（会话首进程）
